# Simulador Web

Aplicación web para proyecciones usando múltiples métodos estadísticos.

## Características

- Proyecciones para 3 meses futuros
- Múltiples métodos (Promedio, Lineal, Monte Carlo)
- Análisis de sobredispersión
- Reportes HTML descargables
- Interfaz responsive

## Formato del Archivo

El archivo debe contener la siguiente estructura:
- AÑO (año del registro)
- MES (nombre del mes en español)
- NUMERADOR (valor numérico)
- DENOMINADOR (valor numérico)

## Despliegue

Esta aplicación está desplegada en Streamlit Cloud.
